Sobre o Projeto

O projeto Pet&Style foi desenvolvido com o objetivo de aplicar conceitos modernos de desenvolvimento front-end utilizando Tailwind CSS. A proposta foi criar uma interface responsiva, organizada e visualmente atrativa para uma loja fictícia de produtos para pets.

 Tecnologias Utilizadas

Durante o desenvolvimento, foram utilizados HTML5 para estruturação da página e Tailwind CSS para estilização, explorando suas classes utilitárias para construção rápida e eficiente da interface.

 Responsividade

O layout foi desenvolvido seguindo a abordagem mobile-first, garantindo que a aplicação funcione corretamente em diferentes tamanhos de tela. Foram utilizados breakpoints como sm, md e lg para adaptação do conteúdo.

Layout e Organização
Foram aplicados conceitos de Flexbox para alinhamento de elementos e CSS Grid para organização da seção de produtos, permitindo uma disposição responsiva e dinâmica dos cards.

Estilização

A estilização foi feita utilizando classes utilitárias do Tailwind CSS, incluindo controle de cores, espaçamentos, tipografia, bordas e sombras, evitando a necessidade de CSS customizado.

 Objetivo

O principal objetivo deste projeto foi consolidar o aprendizado em Tailwind CSS e desenvolvimento responsivo, criando uma base sólida para projetos futuros mais complexos.
